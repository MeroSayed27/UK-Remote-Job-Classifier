# UK Job Classifier

A Python project that analyzes UK remote job postings and classifies them based on user skills.  
This project helps job seekers quickly identify suitable Machine Learning, Data Science, and Data Analyst roles.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Joblib

## Features
- Loads job data from CSV
- Cleans and preprocesses text data
- Combines job title, skills, and description into a single text column
- Trains a Logistic Regression classifier
- Predicts the category of new job postings
- Outputs top recommendations based on user skills
